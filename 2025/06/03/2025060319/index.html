《ArrayList的底层原理》

发表于2025-06-03|更新于2025-06-03
|总字数:187|阅读时长:1分钟|浏览量:

简介

ArrayList 是 Java 中常用的动态数组实现，是基于数组实现的可变长度集合，可以看作是一个动态数组结构，通过一个内部数组 Object[] elementData存储元素。

扩容机制

扩容机制是 ArrayList 的核心，ArrayList 默认的初始容量为 10，当数组容量不够时，会进行扩容。扩容的机制是：将原数组的容量翻倍，并创建一个新的数组，将原数组中的元素复制到新数组中。
